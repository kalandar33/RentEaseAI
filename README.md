## 📌 About the Project

**RentEase AI** is a modern AI-powered Vehicle Rental Management System developed as a **personal portfolio project** to enhance my skills in Full Stack Web Development, Database Management, and Python Integration.

The system enables users to browse vehicles, make bookings, simulate payments, download invoices, and receive intelligent vehicle recommendations using Python.

This project demonstrates practical implementation of modern web technologies and serves as a showcase of my technical capabilities.

# ⚙ Installation

### Clone Repository

```bash
git clone https://github.com/kalandar33/RentEaseAI.git
```

### Move Project

Copy the project folder into

```
xampp/htdocs/
```

### Start XAMPP

Start

- Apache
- MySQL

### Import Database

Create a database named

```
rentease_ai
```

Import the provided SQL file.

### Run

Open

```
http://localhost/RentEaseAI/
```
